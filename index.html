What is Conway's Game of Life?

A cellular automaton devised by the British mathematician John Horton Conway in 1970. One interacts with the Game of Life by creating an initial configuration and observing how it evolves, or, for advanced players, by creating patterns with particular properties.

You can read more about it on the wikipedia entry here.

Rules:

Nodes that are "alive" (blue)

  • 0-1 neighbours: Dies to loneliness.
  • 2-3 neighbours: Survives.
  • 4+ neighbours: Dies to overpopulation.

Nodes that are "dead" (grey)

  • 3 neighbours: Returns to life.
